# Desafios de Programação em C 🚀

Este repositório foi criado para organizar e documentar as soluções dos desafios de programação e algoritmos que vou desenvolvendo ao longo dos meus estudos.

---

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C
* **Ambiente de Desenvolvimento:** Code::Blocks / GCC Compiler
* **Controle de Versão:** Git & GitHub

---

## 📌 Desafios Resolvidos

### 1. Busca de Picos em Matrizes
* **Arquivo:** `busca_picos.c`
* **Descrição:** Um algoritmo eficiente desenvolvido para encontrar "picos" (elementos que são maiores ou iguais aos seus vizinhos) em estruturas de dados matriciais.

### 2. Dimensionamento de Largura de TV
* **Arquivo:** `largura_tv.c`
* **Descrição:** Exercício focado em lógica matemática e condicionais para calcular o dimensionamento ideal de telas baseado em resoluções e proporções.
